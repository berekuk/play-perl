Questhub Markdown syntax

Questhub uses Markdown-based syntax for comments, quest descriptions, and all other player-generated long-form texts.

Headers

Some #'s before a string produces a header:

# Header 1
## Header 2
### Header 3
#### Header 4

Header 1

Header 2

Header 3

Header 4

Formatting

__BOLD__ or **BOLD**

BOLD
_ITALIC_ or *ITALIC*
ITALIC
___BOTH BOLD AND ITALIC___
BOTH BOLD AND ITALIC

Lists

Unordered list

Either + or * or - produces unordered list, marked with round 'bullets':

+ text 1
* text 2
- text 3

  • text 1
  • text 2
  • text 3

Ordered list

1. produces an auto-numbered ordered list:

1. text 1
1. text 2
1. text 3

  1. text 1
  2. text 2
  3. text 3
If you have some text before the list, make sure to preface it with empty line.

Blockquotes

> Level one
>> Level two
>>> Level three
>>>> Level four

Level one

Level two

Level three

Level four

Inline code

Putting `some text` in backquotes produces inline code highlighting like this: some text. You may use it for any purpose you like, not only for code highlighting though.

Horizontal rule

You can make horizontal separation lines using ***.

Links

Links consist of three parts: Link Text, Link Title and URL.
This is inline link template in Markdown: [Link Text](URL "Link Title").

E.g., [QuestHub](http://questhub.io "This is QuestHub!") will produce:

QuestHub

You can omit Link Title to make things shorter.
[QuestHub](http://questhub.io) will produce:

QuestHub

And even shorter by putting URL into <>. So that <http://questhub.io> will make:

http://questhub.io

Finally, you can simply type an URL, and it will be converted to a link :) http://questhub.io:

http://questhub.io

Images

You can put images into your quests. Image code template is the following:
![Alt text](Image_URL "Image Title")

For example, ![QuestHub Chaos](http://questhub.io/i/chaos.png "This is QH Chaos!") makes:

Also you can omit Alt text and Image Title like this:
![](http://questhub.io/i/chaos.png)

And you can combine images with hyperlinks! (Try by pressing the next Flying Spaghetti Monster)
[![](http://questhub.io/i/chaos.png)](http://questhub.io/realm/chaos)

Questhub names highlighting

You can mention any player's name from Questhub using @PlayerName. E.g., @berekuk produces berekuk.

This is not standard feature of Markdown, so be warned. It may not work anywhere else. Also please don't mindlessly mention other players. They will receive e-mail notification telling them that you have mentioned them.

Task lists

Start a line with [ ] or [x] to create a toggleable checkbox. For example:
[x] Did this
[ ] Going to do this

Did this
Going to do this
You can use and manipulate task lists in quests and comments you own. Don't try to use them in stencils and realm descriptions, that's no good.

Final Notes

Most of common html syntax is supported by Markdown parser here.

  • <a href="http://questhub.io">QuestHub</a>
    QuestHub
  • <s>strike-through text</s>
    strike-through text
  • <p>This <br /> is a paragraph <br /> with <br /> line breaks</p>

    This
    is a paragraph
    with
    line breaks

Paragraphs are separated by a single line break, not by double, as in standard Markdown.

Try things for yourself, that's always fun!
